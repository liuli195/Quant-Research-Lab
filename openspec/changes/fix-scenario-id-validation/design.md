## Context

标准量化分析的 JSON Schema（JSON 结构约束）已限制场景标识为 1 至 64 位的小写字母、数字和连字符；本地研究配置加载只检查非空。

## Goals / Non-Goals

**Goals:**

- 在本地研究启动策略和行情读取前拒绝不兼容场景标识。
- 将格式常量放入既有共同结果契约，并用回归锁定其与分析 Schema 的一致性。

**Non-Goals:**

- 不改变分析计划格式、结果包格式或任何公开命令。
- 不修复已生成的历史结果包。

## Decisions

- 在 `result_contract`（共同结果契约）中提供唯一的格式常量和校验函数；本地运行配置加载复用它。该模块已被本地研究使用，避免新建工具层。
- 保留分析 JSON Schema 的声明式校验，并由回归断言其正则与共同契约一致；JSON Schema 不能引用 Python（编程语言）常量。

## Risks / Trade-offs

- [两种表示未来可能漂移] → 回归测试直接比较格式常量和分析 Schema 的规则。
- [历史包仍含旧标识] → 分析入口维持现有拒绝行为，不迁移或改写旧结果。
